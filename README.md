## プロジェクト名

ModelConverter


## プロジェクトについて

BlockbenchのGeneric Modelで作成されたモデルを強制的にModded Entityに変換します。
特に、Per-Face UVで作成されたモデルのBox UVへのコンバートを想定しています。
